ما هو امن المعلومات؟
هو حماية البيانات ورقية او رقمية الخ من الوصول الغير مصرح به سواء هجوم او سرقة او ضرر
البيانات تكون vulnerable بسبب طريقة التخزين او اثناء النقل او طريقة المعالجة
ثلاثة شروط لو تحققت فكذا حققنا امن المعلومات ( CIA ) :
Confidentiality : السرية اي لا يراها الا اشخاص معينين 
Integrity : سلامة البيانات اي لا يقوم بالتعديل عليها الا الاشخاص المصرح لهم
Availability : توافرية البيانات للتعديل او الوصول اليها للاشخاص المصرح لهم 
هناك مبدأ رابع لكنه ليس من الاساسيات : 
Non-Repudiation : وجود شهود او انظمة تكشف اي عملية حصلت بالنظام لئلا يقوم الفاعل بانكار ما فعل 
Security Policy : سياسة الامن او توضيح الاجراءات المتخذة لحماية المعلومات او توضيح الوظائف والمسؤوليات يتم وضعها بواسطة المدراء
وحدات امن المعلومات :
Security Operations Center ( SOC ) : وحدة تحتوي محترفي المجال مهمتها عمل مراقبة وتحليل للمعلومات لاتخاذ اجراء قبل حدوث ضرر فهي تعتبر خط الدفاع الاول 
Incident Response : SOCفريق الاستجابة للحوادث وتارة يكون مندرج تحت الـ
Security Control :CIAحل الذي يقوم بتحقيق الـ
تصنيفات Security Control : 
Technical : هاردوير او سوفتوير مثل الفايروول او الانتي فايروس 
Operational : الافراد المسؤولين عن مهام معينة 
Managerial : الافراد المسؤولين عن ادارة الامن 
وهناك تصنيفات اخرى تعتمد على الهدف مثل : 
Preventive : يشتغل قبل يحصل الهجوم 
Detective : تكشف الهجوم وهو شغال 
Corrective : يعمل تصليحات بعد الهجوم يزيل الهجوم او يقلل منه مثل نظام النسخة الاحتياطية للنظام
Physical : الامور الفيزيائية كاجهزة الانذار او البوابات 
Deterrent : اجراءات تنبيهية مثل اللوحات امام البوابات التي تنبه ان المرور عبر البوابة للمصرح لهم فقط
Compensating : تصحيح او تعويضي عن اجهزة الامان التي فشلت 
ISO Framework : معايير خاصة وجاهزة للمهام المفترض عملها لامن المعلومات

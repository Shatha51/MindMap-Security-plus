Application attack : 
Privilege escalation : يقوم المهاجم برفع الصلاحيات لنفسه 
Remote code execution ( RCE ) : استغلال ثغرة في ابلكيشن عن بعد 
Error handling : ظهور اخطاء من البرنامج تعطي المهاجم معلومات معينة مثلا كادخال كلمة سر خاطئة لحساب الضحية فيظهر له خطا بان كلمة السر اطول فكذا المهاجم اخذ معلومة عن طول الباسوورد
Improper input handling : عدم فحص المعلومات التي يدخلها المهاجم فيقوم المهاجم بوضع معلومات خاطئة حتى يستغل الثغرة 
Buffer overflow ( BOF ) : ادخال بيانات بحجم كبير بلا فائدة لا يستطيع البرنامج معالجتها 
Memory leak and resource exhaustion : اخذ البرامج زيادة للرامات حتى يحصل كراش
DLL injection : حقن ملفات DLL وهي مكتبات من ملفات النظام فكل ما تشتغل المكتبات ذي يتشغل معها المالوير 
Web application attacks : 
Replay attack : لو المهاجم عنده رقم التوكن او الكوكي حقك يقدر يرسل طلب للوجهة برقم التوكن حقك فيدخل بدالك
Token : رقم نحتفظ فيه يغنيني عن ادخال اليوزر والباسوورد كل مرة ويكون موجود في الكوكيز
Session hijacking : سرقة الجلسة عن طريق سرقة الكوكيز في حال هي ما كانت مشفرة وطبعا هو بيكون بنفس الشبكة معاك
Cross-site scripting ( XSS ) : يرسل سكربت من عنده عشان يخرب حاجة بالموقع عن طريق انه يدخل على الموقع ويدور اي حاجة فيها input هو يحطها فيحقن الموقع واي احد يزور الموقع حيصاب بنفس الاتاك 
SQL injection : حقن استعلامات قاعدة البيانات عشان يدخل المهاجم على الموقع وتعرفه على كل محتويات قاعدة البيانات 
How to secure web application? 
Input validation : التحقق من البيانات للتاكد من صحة البيانات وعدم تنفيذ هدمة على الموقع 
Secure techniques : تكنيكات امنة للموقع ومشفرة
